# Form-Validation-
Ce projet est un exemple simple de validation de formulaire en JavaScript.  

<img width="537" height="665" alt="image" src="https://github.com/user-attachments/assets/85e1bdf6-27aa-4ff8-b1ba-8f3ca8f01e82" />

Il permet de vérifier que tous les champs obligatoires sont remplis avant la soumission :  

- Nom et prénom  
- Date de naissance  
- Ville sélectionnée  
- Photo ajoutée  
- Sexe choisi  

Le projet change la couleur des bordures pour indiquer les erreurs (rouge) ou la réussite (vert) et affiche les informations saisies en dessous du formulaire après validation.

## Fonctionnalités
- Validation en temps réel (`blur` et `change`)  
- Messages d’erreur clairs et visibles  
- Validation finale au clic sur "Valider"  
- Réinitialisation du formulaire avec "Vider"  
- Affichage des données saisies de manière lisible
